CSS tutorial

This page is for studying CSS independently. It's a languge used alongside HTML to determin how the text should look like.

CSS format

CSS consists of a property and a value, it's written in this format: value:property;

A paragraph with a different background color.

Font-family defines the family or the type.

font-size defines the size of the text.

font-weight defines the thickness of the text, you can use values such as 100,300,500,700,900 or words like bold

text-align defines the alignment of the text, it can take the values: left/center/right/justify.
